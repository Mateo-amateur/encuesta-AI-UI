  ## Cómo inicializar el proyecto
  Ejecutar el comando:
  ```cmd
pnpm install
```

Y se inicia el puerto con
  ```cmd
pnpm run dev
```

Testing E2E con
  ```cmd
pnpm cypress:open
```
> :warning: Consultar la documentación de Cypress para su instalación

## Requisitos

 - Node.js 18+
 - pnpm habilitado
